# NPC Counter

NPC Counter is a RuneLite plugin that displays a live count of specific NPCs
currently rendered on screen.

## Features
- Counts NPCs by name (comma-separated list)
- Live updating overlay
- Fully draggable overlay
- Customizable font size and color
- Lightweight and performance-friendly

## Use cases
- Slayer tasks
- AFK NPC farming
- Area monitoring (cows, chickens, crabs, etc.)

## Configuration
- **NPC names**: Comma-separated list of NPC names to count
- **Font size**: Adjust overlay text size
- **Color**: Change overlay text color

The overlay updates every game tick and only counts NPCs currently rendered
by the client.
